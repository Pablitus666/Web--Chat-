Términos de Uso

¡Hola! Te damos la bienvenida a Web - Chat. Estos Términos de Uso son como las reglas del juego para que uses nuestra aplicación web. Al usar Web - Chat, aceptas estas reglas. Si no estás de acuerdo con alguna, ¡no te preocupes! Simplemente no uses la aplicación.

1. ¿Quién puede usar Web - Chat?

Web - Chat está disponible para usuarios de todas las edades. No imponemos una restricción de edad para el uso de nuestra herramienta. Sin embargo, recomendamos que los menores de edad utilicen la aplicación bajo la supervisión de un padre o tutor. El uso de cualquier herramienta en línea debe ser siempre responsable. Si utilizas la aplicación en nombre de una empresa, debes tener la autoridad para aceptar estos términos en su representación.

2. ¿Qué es Web - Chat?

Web - Chat es una herramienta sencilla para cifrar y descifrar mensajes de texto. La ofrecemos "tal cual" y "según esté disponible". Esto significa que no podemos prometer que siempre funcionará sin interrupciones o errores, pero hacemos nuestro mejor esfuerzo para que así sea.

3. Lo que puedes y no puedes hacer

3.1. Puedes usar Web - Chat para:

  • Cifrar y descifrar tus mensajes de texto.
  • Usar la aplicación para tus cosas personales, sin fines comerciales.

3.2. Por favor, no uses Web - Chat para:

  • Cualquier cosa ilegal o que no esté permitida.
  • Copiar o usar cosas que no te pertenecen (propiedad intelectual de otros).
  • Enviar mensajes ofensivos, amenazantes o que inciten al odio.
  • Distribuir virus o programas maliciosos.
  • Intentar dañar nuestra aplicación o acceder a ella sin permiso.
  • Intentar "desarmar" la aplicación para ver cómo funciona por dentro.
  • Recopilar información personal de otros usuarios sin su permiso.
  • Hacerte pasar por otra persona o empresa.

4. Nuestra Propiedad

Web - Chat y todo lo que la compone (diseño, funciones, etc.) es propiedad de PABLITUS Inc. Está protegida por leyes de propiedad intelectual. Tú no puedes copiar, modificar o distribuir ninguna parte de nuestra aplicación sin permiso.

5. Lo que no podemos garantizar

Ofrecemos Web - Chat con la mejor intención, pero no podemos garantizar que siempre funcionará perfectamente, sin errores o sin interrupciones. No nos hacemos responsables si algo sale mal o si hay algún problema técnico.

6. Nuestra Responsabilidad (limitada)

PABLITUS Inc. y nuestro equipo no somos responsables de ningún daño indirecto o pérdida que pueda surgir por usar (o no poder usar) Web - Chat. Esto incluye pérdida de datos, ganancias o cualquier otro problema, incluso si nos avisaste de que algo así podría pasar.

7. Si hay problemas (Indemnización)

Si usas Web - Chat de una manera que no está permitida por estos Términos y eso nos causa algún problema legal o financiero, aceptas ayudarnos a defendernos y cubrir los costos (incluidos los honorarios de abogados).

8. Leyes que nos rigen

Estas reglas se interpretan según las leyes del Estado Plurinacional de Bolivia. Si alguna parte de estas reglas no es válida, el resto seguirá siéndolo. Estos Términos son el acuerdo completo entre tú y nosotros sobre el uso de Web - Chat.

9. Cambios en estas reglas

Podemos cambiar estos Términos en cualquier momento. Si el cambio es importante, intentaremos avisarte con al menos 30 días de antelación. Si sigues usando Web - Chat después de los cambios, significa que los aceptas. Si no estás de acuerdo con las nuevas reglas, por favor, deja de usar la aplicación.

Política de Privacidad

Última actualización: 25 de diciembre de 2025

¡Tu privacidad es súper importante para nosotros en Web - Chat! Queremos que sepas exactamente cómo funciona nuestra aplicación en relación con tus datos.

1. No Recopilamos Datos Personales

Así de simple: Web - Chat no recopila ningún tipo de dato personal tuyo. Cuando usas la aplicación, todo lo que haces (cifrar, descifrar mensajes) se procesa directamente en tu navegador. ¡Nada de tu información sale de tu dispositivo!

2. Cero Cookies y Rastreadores

No usamos cookies, píxeles de seguimiento ni ninguna otra tecnología para rastrearte o recopilar información sobre ti. Tu actividad en Web - Chat es completamente privada y anónima.

3. Tus Mensajes son Solo Tuyos

Los mensajes que introduces en Web - Chat nunca se guardan en nuestros servidores ni se transmiten a terceros. Una vez que cierras la aplicación o borras los campos, tus mensajes desaparecen. ¡Tú tienes el control total!

4. Sin Restricciones de Edad (en cuanto a datos)

Dado que no recopilamos ningún dato personal, no hay restricciones de edad específicas en nuestra política de privacidad. Sin embargo, te recordamos que el uso de cualquier herramienta online debe ser siempre responsable.
